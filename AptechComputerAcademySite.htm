Smart Professional Data Science

Aptech AT HSR

Data Science is all about applying data mining techniques,doing statistical analysis,and building high quality prediction systems integrated with organisations products. The responsibilities of a data scientist include data extraction and cleaning,statistical data analyse,present data using tools like Tableau, sas.
With this course, you will learn the most in-demand technologies like Data Science on R, Python, SMAC, Big Data on Hadoop and implement the concepts such as data exploration, regression model, Hadoop, etc



Course details:

Smart Pro Data Science course provided required training in:

Term 1

  • Financial data analysis with MS Excel
  • Python programming
  • Emerging job areas-SMAC
  • Large data management
  • R programming
  • eProject (R),

Term 2

  • Foundation of Big Data systems
  • Processing Big Data [Hadoop-MapReduce, Hive, PigLatin]
  • Visual analytics with Tableau
  • Web & social media analytics (Google Analytics & SAS)
  • Project-Big Data

Course duration:

Smart Pro Data Science is a 14-month* course. Classes are typically held 2 hours a day/ 3 days a week.

Eligibility:

IT students/ professionals/ engineer

Job opportunities:

On successfully completing the Smart Pro Data Science course, you will be ready to join top national & international IT companies as:

  • Data Analyst
  • Data Visualizer
